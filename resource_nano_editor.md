# Using Nano Editor

#### Step 1:  create a file
```bash
nano myfile.py
```

#### Step 2:  add a line in file
```
print("Hello World")
```

#### Step 3:  save file
`ctrl o`  
`<enter>`  

#### Step 4:  exit nano
`ctrl x`  


---
